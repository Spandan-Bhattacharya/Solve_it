How to Play Simon:

1. Simon is a memory game where the goal is to repeat a sequence of colors and sounds as accurately as possible.
2. The game starts with a single color lighting up and producing a sound. The player must repeat the sequence by pressing the corresponding colored buttons.
3. After each successful round, the sequence gets longer and more complex.
4. If the player makes a mistake, the game ends.
5. The game continues until the player can no longer repeat the sequence correctly.
