How to Play Block Bluster:

1. Block Bluster is a puzzle game where the goal is to clear blocks from the board.
2. You clear blocks by selecting groups of two or more adjacent blocks of the same color.
3. Click on a group of blocks to remove them from the board.
4. The more blocks you clear in a single move, the more points you score.
5. The game ends when you can no longer make any moves.
